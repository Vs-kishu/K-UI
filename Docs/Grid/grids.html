Grids

Grid is one of the most powerful tools of CSS, if not the most powerful. It gives the developers so much control over the layouts with so much less lines of code. You can easily create a Grid layout with the help of a few classes, using this library. There are 3 main classes available for the grid layout

  • .gr-col-2 - it creates 2 columns of equal width
  • .gr-col-3 - it creates 3 columns of equal width
  • .grid-autofit - It automatically tries to fit as many columns of the specified width range as it can. There are 3 size ranges available -
  • .gr-autofit-sm for columns of width 12rem to 100%
  • .gr-autofit-md for columns of width 20rem to 100%
  • .gr-autofit-lg for columns of width 30rem to 100%
